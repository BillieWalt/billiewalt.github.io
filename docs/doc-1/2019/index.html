2019

2019-12-31
1 min read

2019，到年底了。

这一年我越发有个感觉，那就是如果你从事一项你所热爱的、与人交流的活动，无论是团学也好，还是社团也好，或是主持人、运动员，那怕是个微博博主什么的，随着时间的推移，表达层面上的技巧都会变得越来越不重要。

那什么是真正重要的呢？真正重要的是你向大家展示的你到底是个怎样的人，你读过的书、看过的电影、听过的音乐、去过的城市、练过的技能、走过的路、甚至爱过的人都会不由自主地从你的表达中弥漫出来，藏都藏不住。

所以我看让别人指道儿，指点迷津，怎么走捷径，这些都不靠谱，经不起实践的检验。

我有时候搞砸了或者做错了事，就反思自己，找张纸来回推演，最后的结论永远指向一点，还是人不行。

柴火捆成大树的样子做不成栋梁，真正能让人可堪大用的就一个办法：扎进土壤，认真生长。

共勉。
